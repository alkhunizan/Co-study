Iconography — Lucide

Halastudy uses Lucide as its primary icon family (1.5px stroke, MIT licensed). Load from CDN once per page:

<script src="https://unpkg.com/lucide@0.469.0/dist/umd/lucide.min.js"></script>
<i data-lucide="camera" style="width:20px;height:20px;color:#A56B43"></i>
<script>lucide.createIcons();</script>

Sizing: 16 (inline), 20 (button), 24 (toolbar), 32 (feature card). Color inherits currentColor. Stroke 1.5px default; 2px above 24px.

Where emoji used to live in Co-Study: migrate 📚 → book-open, 🌧️ → cloud-rain, 🌲 → trees, 🔥 → flame, ☕ → coffee, 🌊 → waves, 🔒 → lock, ⚡ → zap, 🌍 → globe, 🚀 → rocket, 🎤 → mic. Emoji are still allowed in user-generated content (chat, status text, room names).
